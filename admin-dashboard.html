How to Set Up Your Game

Choose Your Game

Select either Quiz Game or Charades from the overview page. Each game type has its own setup flow.

Create a Session

For Quiz: Add questions and choose your theme colors. For Charades: Create categories with flashcards.

Generate Game Code

Once your session is ready, generate a unique game code. This code is reusable and will route players to the correct game.

Share with Players

Share the game code with your players. They can join through the main join page or you can join as a player yourself to test.

Start Playing!

Once players have joined, start the game and enjoy! You can monitor progress, view results, and manage sessions anytime.
